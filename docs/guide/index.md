## 进入服务器

1. 仔细阅读[复读公约](/guide/rules)。
2. 查看你的硬件是否符合 [Minecraft 官网](https://www.minecraft.net/zh-hans/store/minecraft-java-edition#features2)写的最低要求，一般都符合。
3. 参考[入服教程](https://docs.qq.com/doc/DSFpwT1Fra3BMdml1)来进入服务器。
