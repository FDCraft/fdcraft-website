# 数据包列表

## 省流

- 地形与放置的地物（如树，花）有较大变化，增加了群系
- 要塞与村庄有变化，增加了新的结构
- 被沙子砂砾埋住会进虚无之乡，进去找到橡木门或者 spawn 回主世界，诡异木门去微光之境
- 更多的食物
- 家具与手办系统
- 挖宝系统
- 原版变更：袭击恢复到旧版本，砍玩家掉头

## 总览

对于一些没有给出 wiki 的数据包，通常其 Modrinth 界面的简介与画廊足以说明一切。
